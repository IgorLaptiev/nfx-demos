This application demonstrates how you can address different resources such as css,
java scripts, flat files, images etc.

Start the application and enter one of the following URL:
http://localhost:8080/Test/Index or simply http://localhost:8080.

Click one of the following links:
http://localhost:8080/res/files/info.txt?a=true - to download text resource file info.txt.
http://localhost:8080/res/files/data.xml - to see contents of xml resource file data.xml.
http://localhost:8080/res/files/persons.json - to see contents of json resource file persons.json.
All styles and java scripts acted on Index page are downloaded from disk by FileDownloadHandler.

Administrator privileges are required to start the application.
