Test filtering:
1. /bad_url - to show error dump (ErrorFilter in action).
2. /Test/RedirectHome or /Test/RedirectToVideo - to return redirected responce (RedirectFilter in action).
3. /Test/UnexistingAction1 - to redirect to http://localhost:8080/Test/Index (CustomFilter in action).
4. /Test/UnexistingAction2 - to redirect to http://localhost:8080/Test/UnexistingAction1 (CustomFilter in action).>

Test handling:
1. /Test/Box to handle request by Box action of Test controller (MVCHandler in action).
2. /mypath/Test/Box?color=ff7f00 to handle request by Box action of Test controller with parameter(MVCHandler in action).
3. /dump - to see output of current context (ContextDumpHandler in action).
4. /custom/RandomPeople?cnt=1 to see CustomHandler in action.
